Number Formatter
=========

A small library that adds commas to numbers

## Installation

    `npm install @tenorious/number-formatter`

## Usage

    var numFormatter = require('@tenorious/number-formatter');

    var formattedNum = numFormatter(35777);

Output should be `35,777`

## Tests
